# 60-Second Demo Script

0-10s

- Open the repo and say: "This is DarkForest Briefs, a private research CLI built with SolRouter."

10-25s

- Show the command:
- `npm run brief -- --topic "Catalysts for SOL this week" --out ./reports/sol-week.md`

25-40s

- Point out that encryption is enabled by default and the tool can also use live search or BRAID reasoning.

40-55s

- Open the generated markdown report and scroll through:
- `Core Thesis`
- `Catalysts`
- `Risks`
- `Missing Evidence`
- `24h Watchlist`
- `Execution Summary`

55-60s

- Close with: "The point is simple: when the query itself is sensitive, private inference should be built in from the start."
